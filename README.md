# Svelte-todo-app

![image](https://github.com/kimdayeon37/Svelte-todo-app/assets/93921784/96afbd5c-ce45-4f27-ba8b-fcd606c82182)

### 구현 기능 
1. 목록을 추가하면 윗줄에 추가된다. (enter 클릭해도 추가) 
2. 수정버튼과 삭제버튼이 존재
3. 수정버튼을 누르면 ok, cancel 버튼으로 동적 기능 구현 (ok 시, enter 클릭해도 추가)
4. 빈 공백 return
5. 새로고침해도 기존 내용들이 존재
